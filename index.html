Guide

使用说明

  1. 新增人员 在左侧输入学员姓名，点击“新增”。学员卡片会显示已上课时、总学时和剩余学时进度。
  2. 设置资料 选择学员后，在中间设置授课科目、总学时、购买节数和下次上课时间。修改后自动保存到本地缓存。
  3. 增加课时条目 选择上课日期，按小时 H、分钟 M 填写本次课时；详细开始时间默认折叠，可按需填写。记录会按日期与时间自动排序。
  4. 编辑与分析记录 课时可修改、删除或排除分析。排除后的记录会弱化显示，不影响真实余额，也不进入右侧统计文本；还可按日期范围筛选。
  5. 导出 / 导入表格 右侧左列为导出，右列为导入。XLSX / CSV 适合表格查看和交接。
  6. JSON 备份缓存 JSON 是完整备份，包含全部学员、设置和课时。换浏览器、清缓存或部署前，先导出 JSON；恢复时导入 JSON。
  7. 复制反馈文本 右侧会自动生成统计语句和学情反馈模板，点击“复制”即可发送给学生确认。
